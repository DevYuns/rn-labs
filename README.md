# rn-labs

## 🎯 목표
- RN Core 내부 동작 코드 레벨 이해 (JSI, TurboModules, Fabric, Hermes)
- iOS/Android 시스템 아키텍처 숙련 (RunLoop, ARC, Looper, GC, Rendering)
